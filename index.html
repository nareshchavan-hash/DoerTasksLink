Doer Tasks – System Overview

📧 How to Access the System

  • Users receive an email containing the access link
  • Link: https://sites.google.com/superbrealty.in/superbrealty?usp=sharing
  • Open the link → navigate to Doer Tasks → click Checklist
  • Three modules available: Checklist | FMS | Delegation
  • Each module offers three views: List · Calendar · Report
📋 CHECKLIST 🔧 FMS 📤 DELEGATION

Checklist Module – List View

📋 CHECKLIST
🔍 Filters Available
  • Start Date filter
  • End Date filter
  • Frequency / Priority filter
  • Task Name search
📊 Task Summary Tiles
  • Total Tasks (Complete, Pending, Overdue counts)
  • Pending Tasks only
  • Completed Tasks only
  • Overdue Tasks only
⚡ Task Submission Flow
  • Click a task → Google Form opens
  • Submit the form → task auto-updates
  • Status: Pending → Completed (~10 sec)
Superb Realty – Doer Tasks System Confidential

Checklist Module – Calendar View

📋 CHECKLIST
📅 Calendar Overview
  • Tasks shown on a monthly calendar grid
  • Color-coded by status for easy scanning
  • Click any date to filter that day's tasks
🎨 Color Coding
Pending – Awaiting completion
Overdue – Past due date
Done – Successfully completed
💡 Interaction
  • Click a specific date → shows that day's tasks only
  • Visual overview of task distribution across the month
